# 4K去噪材质重绘提示模板

按以下模块组成一段连续中文提示词。根据图像类别替换方括号内容，不遗漏“结构锁定”“背景去噪”“材质再生”和“最终约束”。

## 1. 全场景结构解析

- 图像类别：明确为人物肖像、产品静物、自然风景、建筑空间或其他类别。
- 核心主体：描述主体身份、数量、骨架/姿态、形状、轮廓、关键比例和特殊结构。
- 场景空间：描述背景元素、前后景、透视关系、环境和原始裁切边界。
- 结构锁定：严格保留主体身份、数量、姿态、几何、比例、视角、透视、构图和全部关键设计特征。

## 2. 物理材质脑补与再生

通用要求：彻底忽略参考图中的噪点、脏污、涂抹、塑料感、压缩痕迹和画质损伤；这些不是主体的真实材质。按照主体本该具有的完美物理状态重新生成表面肌理，但不得改变结构。

按类别选用：

- 人物：自然弹性皮肤、真实细微毛孔、自然肤色过渡、单根清晰且不僵硬的头发纹理；保留身份、五官、表情、年龄特征和姿态，禁止蜡像感与过度磨皮。
- 产品：无瑕疵金属拉丝或阳极氧化质感、细腻真实的哑光塑料、高光通透的玻璃、准确橡胶/织物/木材肌理；精准还原原始颜色，清除指纹、灰尘、划痕、污渍与杂质雾感。
- 场景：清晰自然的植物叶片脉络、粗糙真实的混凝土/石材/木材表面、正确的玻璃与金属反射；不得把噪点当作纹理。

## 3. 背景强制去噪

背景与环境必须 clean, pure smooth background, completely noise-free，色调和空间关系保持原图一致；清除颗粒、色块、脏点、条带、压缩痕迹和无意义纹理。不得新增物体、文字、装饰或改变场景。

## 4. 光影、摄影与调性

- 光影：完全延续原图的主光方向、强弱、阴影位置、反射关系和氛围；光线均匀柔和，高光自然通透，无杂乱阴影、光晕和过曝。
- 摄影：保持原始镜头视角、景别、焦距感、透视和景深；极致清晰但不过度锐化。
- 调性：写实、物理可信、干净、无瑕疵，不改变原始配色和视觉风格。

```text

彻底忽略原图中所有劣质涂抹纹理、噪点、脏污、灰尘、划痕、指纹、色块、锯齿、压缩痕迹、塑料感和画质损伤；它们不属于主体。按照主体本该具有的完美物理状态重新生成：[类别专属真实材质描述]。生成内容必须与原图光影质感完全一致，边缘、反射、阴影和透视无缝融合。

背景与环境必须 clean, pure smooth background, completely noise-free，清除全部颗粒、脏点、条带、杂质和无意义纹理，同时严格保持原始色调、空间层次和环境内容。

延续原图主光方向、明暗关系、阴影位置、反射关系、镜头视角、焦距感、景深和整体氛围；光线均匀柔和，高光自然通透，立体感真实，极致清晰但不过度锐化。

强制约束：只重新生成劣质表面纹理和画质损伤；不得改动主体结构、身份、数量、姿态、比例、关键细节、颜色、背景布局、视角、透视、构图或裁切；不得新增文字、水印、人物、物体、装饰；不得出现重复结构、结构漂移、AI伪影、过度磨皮、蜡像感、塑料感、噪点、脏污、光晕或插画感。保持与原图相同长宽比例，为最终真实4K输出保留完整画面。
```
